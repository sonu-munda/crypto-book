# Cryptotrack

Cryptotrack is a cryptocurrency tracking platform that enables users to monitor cryptocurrency.

## Getting Started

Project Structure
--
```
Lib
|
|------API
|------Bloc-|
|           |--Event, States
|           |--CryptoListBloc, CryptoDetailBloc
|
|------Models
|------Repositories
|------Pages
|------Widgets (Ui Components)
|------Utils
```

Install Dependency
---
```
flutter pub get
```

Run Project
---
```
flutter run
```

Build Apk
---
```
flutter build apk
```
